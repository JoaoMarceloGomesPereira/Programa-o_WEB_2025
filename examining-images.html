Image Sources

Para encontrar os URLs das imagens utilizadas, eu procurei imagens públicas disponíveis na internet. Algumas delas vieram de bancos de imagens gratuitos, como o Freepik, enquanto outras foram obtidas do W3Schools, que fornece imagens de exemplo para testes em HTML. Para obter o URL de uma imagem, basta clicar com o botão direito sobre a imagem desejada em um site e selecionar "Copiar endereço da imagem". Esse link pode ser utilizado diretamente no atributo src da tag . Na tag , utilizei os seguintes atributos: src: Define o caminho da imagem (seja local ou via URL). alt: Texto alternativo exibido caso a imagem não carregue, útil para acessibilidade. width e height: Ajustam o tamanho da imagem (neste caso, width="250px" e height="auto" para manter a proporção). Além disso, usei CSS para adicionar bordas, sombras e um layout flexível, garantindo uma boa apresentação das imagens na página.
